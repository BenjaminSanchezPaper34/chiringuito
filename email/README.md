# Mailing ouverture 2026

## Fichiers
- `ouverture-2026.html` — Version HTML compatible tous clients mail
- `ouverture-2026.txt` — Version texte plain (fallback automatique)

## Comment importer dans Zenchef

1. Connecte-toi à ton dashboard Zenchef
2. Section **Marketing → Campagnes email → Nouvelle campagne**
3. Choisis "Code HTML personnalisé"
4. Colle le contenu de `ouverture-2026.html`
5. Configure le destinataire : tous les clients
6. Personnalise l'objet (voir suggestions ci-dessous)

## Objets d'email à tester (A/B)

**Élégants** :
- "Le Chiringuito ouvre vendredi 8 mai"
- "Saison 2026 : on vous retrouve le 8 mai"
- "Vendredi 8 mai · Le Chiringuito vous attend"

**Plus accrocheurs** :
- "🏖️ La saison commence vendredi"
- "C'est officiel : ouverture le vendredi 8 mai"
- "Réservez votre table pour notre ouverture"

**Avec urgence** :
- "Premier jour d'ouverture : réservez votre table"
- "Vendredi 8 mai · les premières réservations partent vite"

## Pré-header (preview text)
"Rendez-vous vendredi 8 mai pour l'ouverture de notre 11ème saison. Réservez dès maintenant votre table ou votre transat."

## Bonnes pratiques d'envoi

- **Jour idéal** : Mardi ou jeudi, 10h-11h ou 17h-18h
- **2 envois recommandés** :
  1. J-15 : annonce ouverture + réservez (24 avril)
  2. J-3 : reminder week-end ouverture (5 mai)
- **Variable Zenchef** `*|UNSUB|*` à remplacer par le lien de désinscription auto

## Tracking
Zenchef te donnera automatiquement :
- Taux d'ouverture (objectif > 30%)
- Taux de clic sur les CTAs (objectif > 5%)
- Conversions en réservation
